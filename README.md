# yizoui

yizoui 是个人制作的测试 ui 库

## 安装

使用 npm 或 yarn 安装

```
$ npm install yihzo-firstui --save
```

## 导入组件

```
import {Button} from 'yihzo-firstui';
```

## 在线文档

https://yulongcs.github.io/yizoUi/

<br />

## 本地文档

下载代码，npm 安装，使用`npm run storybook`即可获得本地文档。

```
git clone https://github.com/Yizo/yizoUi.git
npm install
npm run storybook
```
